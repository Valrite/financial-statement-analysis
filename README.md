# Financial Statement Analysis

## Project Overview

This project analyzes financial data of Indian companies using Python and Jupyter Notebook. The analysis focuses on market capitalization, quarterly sales performance, category-wise comparison, correlation analysis, and business insights.

## Dataset Features

- Name
- Mar Cap - Crore
- Sales Qtr - Crore
- Market_Cap_Category
- Sales_Qrt_Category

## Objectives

- Clean and preprocess financial data
- Analyze market capitalization distribution
- Analyze quarterly sales performance
- Identify top-performing companies
- Study the relationship between sales and market value
- Generate business insights and risk observations

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- VS Code

## Analysis Performed

### Data Cleaning
- Missing value handling
- Duplicate removal
- Data validation

### Exploratory Data Analysis
- Top companies by Market Capitalization
- Top companies by Quarterly Sales
- Distribution Analysis
- Outlier Detection
- Correlation Analysis

### Business Metrics
- Average Sales by Category
- Sales Efficiency Ratio
- Market Cap Comparison
- Risk Assessment

## Key Insights

- Market capitalization is highly concentrated among a few companies.
- Quarterly sales show significant variation across companies.
- Strong positive relationship exists between market value and sales performance.
- Several companies appear highly valued relative to their sales figures.
- Presence of extreme outliers affects overall averages.
